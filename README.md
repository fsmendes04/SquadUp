# SquadUp

**SquadUp** is the only app your friend squad will ever need:  
Split expenses, plan events, make silly bets, chat, compete in rankings, and keep the memories — all in one beautiful, fast, cross-platform app.

No more switching between Splitwise, WhatsApp, Facebook Events, Google Forms polls, and “who still owes me money?”  
SquadUp does it all. For real friends only.

## Core Features

### 1. Expense Splitting (No more drama)
- Add expenses with receipt photo  
- Smart splitting: equal, custom amounts, percentages, or shares  
- Real-time balances → instantly see who owes whom  
- Debt simplification: minimum transactions to settle up  
- Payment links (MBWay, PayPal, Revolut – coming soon)

### 2. Activity Feed / Mural
- Post text, photos, videos & stories  
- “Moments” – highlight trips, parties, or legendary nights  

### 3. Fun Bets & Predictions
- Create bets: “Who pays the next round?” / “Will Pedro be late again?”  
- Friends vote & pick winners  
- Automatic points, medals, and bragging rights  
- Leaderboard per group and all-time

### 4. Events & Plans
- Create events with date, time, location & cover photo  
- Push notifications + export to calendar

### 5. Group & Private Chat
- Real-time chat (1-on-1 and group)  
- Link messages to expenses, events, or bets  
- Voice notes, GIFs, replies, message reactions  
- Powered by Supabase Realtime / WebSocket

### 6. Rankings & Gamification
- Custom rankings created by the group  

### 7. Profile & Stats
- Avatar, nickname, bio  
- Personal dashboard with all stats
